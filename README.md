### Технологический стек проекта

**Архитектура и сборка**
*   Vite 7.1.7 – современный и быстрый инструмент для сборки и разработки.
*   TypeScript 5.9.3 – для статической типизации и повышения надёжности кода.
*   React 19.1.1 & React DOM 19.1.1 – библиотеки для построения пользовательского интерфейса и его рендеринга.

**Маршрутизация**
*   React Router DOM 7.9.4 – организация клиентской маршрутизации для SPA.

**Управление состоянием**
*   Redux Toolkit 2.9.2 & React-Redux 9.2.0 – современный стейт-менеджер с оптимизированным API для работы с состоянием приложения.

**Работа с API**
*   Axios 1.12.2 – HTTP-клиент для выполнения запросов к серверу.

**Пользовательский интерфейс и стили**
*   React Slick 0.31.0 & Slick Carousel 1.8.1 – компоненты для создания каруселей и слайдеров.
*   Classnames 2.5.1 – утилита для условного объединения CSS-классов.
*   Date-fns 4.1.0 – библиотека для удобной работы с датами и временем.

**Drag-and-Drop**
*   React DnD 16.0.1 – система для реализации интерактивного перетаскивания элементов.

**Специализированные компоненты**
*   QRCode React 4.2.0 – генератор QR-кодов.

**Инструменты разработки**
*   ESLint 9.36.0 – линтер для анализа кода.
*   TypeScript ESLint, ESLint Plugin React Hooks, ESLint Plugin React Refresh – расширенные правила линтинга для TypeScript и React.

### Ключевые функциональные возможности

**Управление кинопоказом**
*   Полный цикл работы с фильмами: добавление (с загрузкой постера, указанием названия, длительности, страны и описания), валидация и удаление.
*   Визуальное планирование расписания с помощью Drag-and-Drop для распределения фильмов по залам.
*   Цветовая индикация сеансов в расписании.

**Конфигурация кинозалов**
*   Создание и удаление залов с уникальными названиями.
*   Гибкий визуальный редактор схемы зала: настройка рядов и мест (до 20×20), разметка типов мест (обычные, VIP, заблокированные).
*   Управление продажами: возможность включать/отключать продажи для каждого зала и устанавливать индивидуальные цены на разные типы мест.

**Управление сеансами**
*   Интерактивное Drag-and-Drop расписание для назначения фильмов и времени показа.
*   Автоматическая проверка на пересечение сеансов с учётом продолжительности фильма.
*   Удаление сеансов с обязательным подтверждением.

**Система онлайн-бронирования**
*   Интерактивная схема зала с подсветкой доступных, выбранных и занятых мест.
*   Динамический расчёт стоимости с отображением цены для разных типов мест.
*   Защита от бронирования уже занятых мест.
*   Оформление заказа с формированием QR-билета, содержащего всю информацию о сеансе (фильм, время, места).

**Безопасность и доступ**
*   Авторизация администратора по email и паролю.
*   Защищённые маршруты: неавторизованные пользователи перенаправляются на страницу входа.
*   Возможность безопасного выхода из системы.

**Дополнительные функции**
*   Календарь сеансов на две недели вперёд с подсветкой текущей даты и выходных.
*   Генерация QR-билетов в SVG-формате.
*   Полностью адаптивный интерфейс, оптимизированный для всех устройств.
*   Единое глобальное состояние приложения (Redux), сохраняющееся между перезагрузками страницы.

**Пользовательский опыт (UX)**
*   Визуальные подсказки и плавные анимации.
*   Интуитивная навигация со сворачиваемыми разделами в админ-панели.
*   Чёткие и понятные сообщения об ошибках и состояниях загрузки.
*   Быстрые действия для отмены операций.
